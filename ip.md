```sh
ip 
```

## subcommand
- addr  디바이스 주소 리스트
- link  디바이스 업다운등
```sh
ip link set [eth0] up
ip l[ink] set [eth0] down
```

## option
- `-br` brief
- `-c`  color
- `-4`  filter ipv4
- `-6`  filter ipv6
- `-s`  show statistics
