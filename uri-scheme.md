# uri-scheme

- https://www.npmjs.com/package/uri-scheme

```sh
npx uri-scheme list
npx uri-scheme add <your scheme>
npx uri-scheme remove <your scheme>
npx uri-scheme open <your scheme>
```

```sh
npx uri-scheme open http://expo.dev --android
```

## options
- `a` android
- `i` ios
