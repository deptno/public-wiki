# console

## console.dir
```sh
console.dir(object, {
  maxArrayLength: 1
})
```
array 를 로그로 찍을때 하나만 찍어줘서 보기 편하게 보여준다
```sh
  Count: 13370,
  Items: [
    { category: [Object], rk: [Object], hk: [Object] },
    ... 13369 more items
  ],
```

- 다른 로그 라이브러리를 사용하거나 프린트할 텍스트만 만들때는 `node:util` 의 `inspect` 메소드를 참조한다
