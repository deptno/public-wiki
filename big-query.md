# big-query
> firebase 와 연동하여 데이터쿼리를 가능하게한다

- 무료는 60일 용량 10GB, 쿼리 1TB 제한
- 생성시 firebase 콘솔의 프로젝트 설정에서 Big Query 를 연결
  - 옵션은 모두 켜줌
  - region 은 `asia-northeast3` 가 한국이라 그렇게 설정, Crashltics 는`US` 고정
  - 연동가능한 서비스
    - Google Analytics
    - Crashltics 
    - Performance Monitoring
    - Cloud Messaging
