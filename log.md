# log

## usage
- 조합은 [[pino]] 와 [[tailspin]] 을 조합하여 사용
- pino-pretty 를 통해서 보기 편하게 뽑고 거기에 [[tailspin]] 을 통해 색을 더하는 구조
```sh 
node program.js | npx pino-pretty -L trace | tspin
```
